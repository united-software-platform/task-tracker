# Контекст проекта

Заполни этот файл описанием технического стека твоего RAG-проекта. Навык использует его при составлении ТЗ и аудите.

Рекомендуемые разделы:
- Основной стек (RAG framework, embeddings, vector DB, LLM)
- Форматы документов и библиотеки обработки
- Ключевые конфигурационные файлы
- Архитектурные решения и их обоснования
- Зависимости при изменении компонентов
